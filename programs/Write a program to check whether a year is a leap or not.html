AIM: Write a program to check whether a year is leap or not.
ALGORITHM:
        Step 1: Start.
        Step 2: Declare & Read variable year.
        Step 3: Check if year%4==0 || year%400==0&&year%100==0 then: Display leap year.
        Step 4: Else: Display Not a lear year.
        Step 5: Stop
CODE:
    #include <stdio.h>
    #include <iostream.h>
    #include <conio.h>
    void main()
    {
        int n;

        }
