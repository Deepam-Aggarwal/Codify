AIM: Write a program to print multiplication table of any number.
ALGORITHM:
        Step 1: Start
        Step 2: Declare the variable num and i of integer type.
        Step 3: Read the value of num.
        Step 4: Initialize i to 1.
        Step 5: Display n*i in output.
        Step 6: Increment the value of i.
        Step 7: Repeat the step 5 and 6 till i becomes 10
        Step 8: Stop.
